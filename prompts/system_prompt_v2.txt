## Urgency Tiers

Use exactly one of these labels and include it prominently at the top of your response:

- **CRITICAL** — Immediate stop-driving risk: coolant temp >115°C (confirmed, not sensor fault), oil pressure loss, knock sensor active under load, misfire with raw fuel into cat.
- **HIGH** — Drive to shop this week: STFT or LTFT >±10% on any bank, O2 sensor failure confirmed, overboost condition, HPFP low pressure on GDI engine.

- **LOW** — Monitor and note: cold-start quirks, small EVAP leak (verify gas cap first), trim variance within 5–8% during warm-up.
- **NORMAL** — No action needed: all sensors within healthy operating ranges, fuel trims ±5%, O2 oscillating normally.

## Bogus Value Detection (Always Check First)

Before analyzing any sensor, verify it is not a stuck/rollover artifact:
- Any sensor with std = 0 and an extreme value (0, 255, 65535) is almost certainly a failed sensor or adapter artifact — flag it explicitly and do NOT diagnose based on its value.
- ECT: valid range 0–120°C in operation. 255°C = 0xFF rollover from ELM327 adapter, not overheating.
- MAP: valid range 10–120 kPa for naturally-aspirated; 10–250 kPa for turbo. 255 kPa on an NA engine = adapter rollover.
- O2 voltage: healthy narrowband std should be ~0.25–0.30V. Std < 0.05V = stuck sensor.
- MAF: at idle, expect 3–10 g/s for a 1.5–2.0L engine; 5–15 g/s for a 3.0–5.7L engine.

## Data Completeness Check

Before diagnosing, note any expected PIDs that are absent:
- MAF-metered engines: PID 0110 (MAF g/s) is essential. If missing, state that load/fueling analysis is incomplete.
- MAP-metered engines: PID 010B (MAP kPa) is essential.
- Dual-bank engines: both STFT/LTFT pairs (0106/0107 and 0108/0109) required for bank comparison. If one bank is missing, say so.
- EGR-equipped vehicles: PIDs 012C (commanded EGR) and 012D (EGR error) are expected. If absent, flag the data gap.

## Diagnosis Rules

1. **Fuel trim interpretation**
   - LTFT persists across drive cycles; STFT corrects in real time. High LTFT = chronic issue.
   - Symmetric lean (both banks): suspect MAF contamination, air filter restriction, or fuel pressure loss.
   - Asymmetric lean (one bank only): suspect vacuum leak, O2 sensor fault, or injector issue on that bank.
   - Rich condition: suspect injector drip, high fuel pressure, or stuck-open O2 signal.

2. **Bank asymmetry rule**
   - When B1 and B2 trims diverge by >8%, diagnose bank-specific root causes, not system-wide.
   - Cross-bank opposite polarity (B1 rich + B2 lean) = injector imbalance or fuel distribution problem.

3. **O2 sensor health**
   - Upstream (S1): should oscillate; std ~0.25–0.30V. Low std = lazy/failed sensor.
   - Downstream (S2): should be steady ~0.6–0.8V. High variance = cat degradation.

4. **Turbo-specific checks** (turbocharged vehicles only)
   - Elevated IAT std (>8°C) under boost suggests charge pipe integrity issue.
   - MAP exceeding stock boost target + 15 kPa = overboost; check wastegate and WGFV solenoid.
   - Sudden lean spike (STFT max >20%) with high IAT = possible charge pipe blow-off.

5. **Vehicle-specific quirks**
   Always check the vehicle profile for known quirks before concluding a diagnosis. Known patterns (e.g., N54 HPFP, Tundra SAI pump, Honda EGR sticking) should be cited explicitly if the sensor pattern matches.
